Export product in CJ Affiliate system. 

== Description ==

CJ Affiliate by Conversant (formerly Commission Junction) is the leading global affiliate marketing network, specializing in pay-for-performance programs that drive results for businesses around the world. 
Our plugin helps you to connect WordPress website to CJ Affiliate. The main task of the plugin is to transfer (export) data to CJ system. 
At the moment you can choose export file format (XML, CSV) and  we can offer you export PRODUCTS through three methods.
1) Via FTP. 
2) Via Email.
3) Download export file. 

Currently we support the following Data in file: 
1.name
2.keywords
3.description
4.sku
5.buyurl
6.price
7.currency
8.startdate
9.instock
10.available
11.imageurl
12.advertisercategory


Notice. 
If you want new data added in export file, please write us and we will do this!


== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Tools->CJ Export screen to configure the plugin
